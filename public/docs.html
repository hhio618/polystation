Get started

    API Endpoint

        https://api.defistation.io/

Defistation API provides you access to TVL data of DeFi projects listed on defistation.io

To use this API, you must be issued a user name and API key. Please contact us at defistation@stamper.network to get your own API key.

get defiNames


# Here is a curl example
curl -X GET \
  https://api.defistation.io/defiNames \
  -H 'Authorization: Basic XxXxXxXxXxpmMjQ5YmQyMC0zMzkzLTExZWItYmVmZC0yMzRiODhkMjNhMWE='

To get definames, make a GET call to the following URL:
https://api.defistation.io/defiNames

Errors

The Defistation API uses the following error Messages:

Error Messages Meaning
[Unauthorized] ... Unvalid Authorization key for the api. This error appears if you didn't input proper Authorization key.
